* Visão Geral do Projeto

- *Nome*: gepisopendata
- *Tipo*: Aplicativo desktop usando Tauri v2 (Release Candidate)
- *Frontend*: Angular 20.3.0
- *Backend*: Rust
- *Plataforma*: Multiplataforma (Linux, Windows, macOS via Tauri)
- *Versão*: 0.1.0


* Arquitetura

** Pilha Tecnológica
- *Framework Desktop*: Tauri v2 (RC) - Framework baseado em Rust para aplicativos desktop
- *Frontend*: Angular 20 com TypeScript
- *Backend*: Rust com runtime assíncrono (Tokio)
- *Cliente HTTP*: Reqwest 0.12 (cliente HTTP assíncrono)
- *Suporte a Containers*: Bollard 0.19.4 (cliente da API Docker)
- *Configuração*: Suporte YAML via serde_yaml
- *Ferramenta de Build*: Cargo (Rust)


** Plugins Ativados do Tauri
- ~tauri-plugin-shell~ - Execução de comandos shell
- ~tauri-plugin-http~ - Requisições HTTP
- ~tauri-plugin-fs~ - Operações de sistema de arquivos

* Padrões de Desenvolvimento

** Comunicação Frontend-Backend

- Usa o plugin HTTP do Tauri para chamadas à API backend
- Serviços interface com o sistema de configuração Tauri
- Dependência de configuração via ambiente (arquivos .env presentes)


** Gerenciamento de Configuração

- Arquivos ~.env~ e ~.env-template~ presentes
- Suporte a configuração em YAML no backend
- Configuração baseada em serviços (tauri-config.service, web-config.service)


** Qualidade de Código

- Prettier configurado: largura 100 caracteres, aspas simples
- Parser Angular para arquivos HTML
- TypeScript em modo strict
- Suporte a testes do Angular CLI


* Recursos Gerados

- acl-manifests.json

- linux-schema.json


* Observações Importantes

1. *Integração Docker*: Dependência Bollard sugere que a aplicação gerencia/interage com containers Docker

2. *Suporte Multiplataforma*: Tauri empacota com suporte a RPM, DEB e geração de ícones para todas as plataformas

3. *Versão de Ponta*: Tauri v2 RC - versão inicial/experimental

4. *Build Concorrente*: Usa ~concurrently~ para builds paralelos de Angular + Tauri

5. *Orientado a Configuração*: Ênfase em configuração por ambiente e baseada em serviços

6. *Validação de Esquema*: Esquemas JSON sugerem validação de dados estruturados


* Possível Caso de Uso

Aplicação de dashboard/administrador para gerenciar "GEPIS" (possivelmente sistema de dados geoespaciais) com capacidades de orquestração de containers, usando Tauri para entrega multiplataforma em desktop.
